sdpctl

sdpctl is a command line tool to manage AppGate SDP Collectives

Synopsis

The official CLI tool for managing your Collective.

Options

      --api-version int      Peer API version override
      --ci-mode              Log to stderr instead of file and disable progress-bars
      --debug                Enable debug logging
      --events-path string   send logs to unix domain socket path
  -h, --help                 help for sdpctl
      --no-interactive       Suppress interactive prompt with auto accept
      --no-verify            Don't verify TLS on for the given command, overriding settings from config file
  -p, --profile string       Profile configuration to use

SEE ALSO

  • sdpctl admin-messages - List of all Admin Messages generated by the system

  • sdpctl appliance - Manage the appliances and perform tasks such as backups, upgrades, metrics etc

  • sdpctl completion - Generate shell completion scripts

  • sdpctl configure - Configure your Collective

  • sdpctl device - Perform actions on registered devices and their tokens

  • sdpctl open - Open the Admin UI in your browser

  • sdpctl privileges - show your current user privileges

  • sdpctl profile - Manage configuration for multiple admin profiles

  • sdpctl service-users - Manage Service Users

  • sdpctl sites - Root command for listing and checking status of sites in a collective
