sdpctl device list

List distinguished names of active devices

Synopsis

List distinguished names of active devices, either in table format or JSON format using the ‘–json’ flag

sdpctl device list [flags]

Examples

  # default list command
  > sdpctl device list

  # print list in JSON format
  > sdpctl device list --json

Options

      --descending         Reverses the order of the device list
  -h, --help               help for list
      --order-by strings   Order devices list by keyword. Available keywords are 'distinguished-name', 'hostname', 'username', 'provider-name', 'device-id' and 'username' (default [distinguished-name])
